Cara import ke Contact / CRM

  1. Google Contacts: Import CSV atau .vcf.
  2. iPhone: Buka .vcf → Add to Contacts.
  3. Android: Contacts → Import from .vcf.
  4. CRM/Airtable/Notion: Import CSV.

Nota: Semua proses client-side. Untuk OCR lebih tepat, boleh integrasi Google Vision / AWS Textract.
